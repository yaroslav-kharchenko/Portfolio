Чтобы запустить СПА нужно выполнить следующие шаги:
1). Выполнить npm i в корне
2). Запустить сервер командой npm start
3). Зайти на http://localhost:8081/
4). Пройти авторизацию по данным: 
        login: test@gmail.com 
        password: 123456
        
        Пустой пользователь:
        login: empty@gmail.com
        password: 123456
